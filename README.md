# Reminder App (UI)

A simple reminder through Email app to leave in time to your destination (UI part)
For server side code, go [here](https://github.com/shubhammxv/reminder-app-back)

## Technologies used

- React.js
- React redux
- JavaScript
- HTML/CSS
- NPM

## Build Setup

``` bash
# install dependencies
$ npm install

# launch server
$ npm start
```
